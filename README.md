# Protos-main-service-tournaments Repository

![Go](https://img.shields.io/badge/go-1.22-blue)
![Protobuf](https://img.shields.io/badge/protobuf-3.21-blue)
![Status](https://img.shields.io/badge/status-active-brightgreen)

Репозиторий содержит **Protobuf файлы** и сгенерированные пакеты для микросервиса турниров.
Сгенерированные файлы Go (`*.pb.go`) находятся в соответствующих папках рядом с proto-файлами для удобного использования напрямую в проектах.

---

## Структура

- `proto/tournaments/` — proto файлы
- `gen/go/tournaments/` - сгенерированный код
